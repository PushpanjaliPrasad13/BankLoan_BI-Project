# Project Overview

Financial institutions need clear visibility into their loan portfolio performance, borrower risk, and repayment behavior.
This project analyzes bank loan data and builds an interactive Power BI dashboard to monitor loan applications, funded amounts, repayment performance, and credit risk indicators.

The dashboard enables stakeholders to quickly identify good vs bad loans, lending trends, borrower profiles, and regional performance, helping banks make data-driven lending decisions.

# Business Problem

Banks process thousands of loan applications, but without proper analytics it becomes difficult to:

Track loan approval trends
Monitor repayment performance
Identify high-risk borrowers
Analyze regional lending distribution
Understand customer borrowing behavior

The dashboard solves these problems by providing real-time insights into the loan portfolio.

# Key KPIs Tracked
#  Core Lending KPIs
•	Total Loan Applications
•	Total Funded Amount
•	Total Amount Received
•	 Average Interest Rate
•	 Average Debt-to-Income Ratio (DTI)

# Loan Quality Metrics
•	Good Loan Applications
•	Bad Loan Applications
•	Good Loan Percentage
•	Bad Loan Percentage

These metrics help measure loan portfolio health and credit risk exposure.

# Dashboard Pages
# 1.	Summary Dashboard
      High-level overview of loan performance.
      Includes:
        •	KPI Cards
        •	Good vs Bad Loan Distribution
        •	Funded vs Received Amount
        •	Loan Status Breakdown
# Purpose:
  Provides executive level overview of loan portfolio health.

 # 2. Overview Dashboard
      Analyzes lending patterns using:
        •	Monthly Loan Application Trends
        •	Loan Distribution by State
        •	Loan Term Analysis
        •	Employment Length Analysis
        •	Home Ownership Distribution
# Purpose:
Helps understand borrower demographics and lending patterns.

# 3. Loan Details Dashboard
      Provides record-level loan insights including:
        •	Loan ID
        •	Loan Amount
        •	Interest Rate
        •	Loan Status
        •	Loan Purpose
        •	Borrower Information
# Purpose:
Allows analysts to drill down into individual loan records.

# Key Insights
  •	Majority of loans fall under Good Loan category, indicating strong portfolio performance.
  •	Debt Consolidation is the most common loan purpose.
  •	Certain states contribute significantly to total loan funding.
  •	Borrowers with longer employment history tend to have better repayment behavior.
  •	Monitoring Debt-to-Income Ratio helps identify high-risk borrowers.
